# tools-compare-dir-files

### Description
This script compare files in directories

### Input Arguments

| Name | Description |Example|
|---|---|---|
|DIR_FROM|Branch from||
|DIR_TO|Branch to||

### Result
Result consisit all files (modified, unmodified, deleted, new).
Each line is beginned from symbol:

| Symbol | Description |
|---|---|
| - | String removed |
| + | String added  |
